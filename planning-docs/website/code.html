API Endpoints

RezNet AI exposes several RESTful endpoints for interacting with the core system. Authentication is handled via encrypted data packets. Refer to the /docs directory for full specifications.

  • POST /api/v1/data-stream - Ingest real-time data.
  • GET /api/v1/analytics - Retrieve processed analytics.
  • PUT /api/v1/protocol/:id - Update an existing learning protocol.
